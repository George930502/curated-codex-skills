# Support

Use [GitHub Discussions](https://github.com/George930502/curated-codex-skills/discussions)
for usage questions and workflow ideas. Use the bug issue form for reproducible
repository defects. Security and conduct reports follow their dedicated private
channels.

Include the operating system, shell, Codex version, install command, validation
output, and the smallest reproduction. For a blocked native-input workflow,
also include `codex features list` after removing machine-specific or sensitive
configuration. Start with
[`docs/windows-native-input.md`](docs/windows-native-input.md) on Windows.

Support is best-effort. This project supports the environments and surfaces
listed in the README; it cannot guarantee availability of experimental Codex
features in every client or account.
