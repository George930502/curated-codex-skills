# Governance

Curated Codex Skills is currently maintained by
[@George930502](https://github.com/George930502). The maintainer owns releases,
security response, repository settings, and final decisions on catalog scope.

Decisions favor observable user value, a small maintenance surface, preserved
provenance, and deterministic validation. Significant behavior or governance
changes should begin in a public issue and record the rationale. Security and
conduct matters remain private.

Contributors earn trust through sustained, constructive work. Additional
maintainers may be appointed publicly with explicit repository scope. If the
project becomes unmaintained, the maintainer should announce that status and
seek a responsible successor rather than silently accepting unsupported work.
